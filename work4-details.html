주요업무 및 업무성과 결과

알루미늄코일 자동화 창고관리시스템(WMS) 개발

  • 개발기간 : 2021. 03 ~ 2021. 10
  • 역할 및 기여도 : 풀스택 개발자 (40%)
기여도 100%

주요업무 및 업무성과

  • 초기 프로젝트 구조 설정
  • ERWin을 이용하여 데이터모델링 된 ERD에서 Oracle의 DB Table 생성스크립트 추출
  • 생성한 DB Table 기반으로 Spring MVC 구조인 vo, dao, sql 파일을 생성
  • 설계된 서비스들을 토대로 기본 페이지들을 생성하여 초기 프로젝트 설정
  • 관리자 화면 구현, MES 연계 테스트 준비
  • ECS와 연동으로 수동 입고 및 출고, 창고 상태 및 관리 기능 등 구현
  • 대시보드에서 SVG 이미지를 제어하여 작업 현황과 각 장비의 상태 모니터링 구현
  • RabbitMQ 관리자 페이지에서 MES-WMS 간의 메시징 처리 및 응답 기능에 대한 자체 테스트
  • 현장 시스템 운영 테스트, MES와의 연계 테스트와 모니터링과 이슈사항 대응
  • 2021년 9월부터 10월까지 현장(해외)에 6주간 상주
  • 현장에 상주하며 WMS 운영 테스트와 이슈사항 대응
  • 고객사 요청으로 창고 내 6개 영역에 물품을 구분하여 지정하는 기능 추가, 영향 범위의 기능 수정
  • 현장에서 MES측과 연계하여 시나리오 운영 테스트 진행, 시나리오 및 메시지 등 관련 이슈 보완
  • Abnormal Cases 대한 대응 시나리오 협의와 기능 보완, 테스트 진행
  • 현장 관리자와 작업자들 대상으로 시스템 매뉴얼 교육 진행

결과

  • 기존에 설계한 WMS 모든 기능 100% 개발 완료
  • MES 연계 시나리오 테스트, 단위 테스트 등 WMS 기능과 연관된 테스트 및 보완 100% 완료
  • 주요 설비의 설치 일정 연기로 전체 운영 테스트 일정은 원격을 통해 국내에서 모니터링 진행
  • 고객사 요청 건과 모니터링 중 발생된 WMS 관련 이슈사항 100% 처리 완료 후 현장 철수
