주요업무 및 업무성과 개선작업 및 성과 결과

월드컵 특집 페이지 개발

  • 개발기간 : 2021. 04 ~ 2021. 12
  • 역할 및 기여도 : 풀스택 개발자 (100%)
기여도 100%

주요업무 및 업무성과

  • 이전에 개발한 특집 페이지 프로젝트 내에서 신규 특집 페이지 화면 구현
  • 기존의 프로젝트 구조 파악 후 분기 처리 작업으로 신규 특집 페이지 생성
  • Interceptor에 신규 특집 페이지를 호출되도록 URL 도메인 분기 작업
  • 소스 관리/배포 툴을 활용하여 내부 소스 관리와 고객사로부터의 피드백 수용
  • TortoiseSVN 연동으로 사내에서 별도 소스 관리
  • GitLab과 Jenkins 연동으로 고객사의 피드백 수용과 오픈 이후 발생되는 이슈들을 빠르게 대응
  • API 데이터 기반으로 각종 미디어 자료 렌더링 작업
  • 플레이 영상 데이터는 JAVA를 사용하여 API 호출 작업 수행
  • 영상 외 이미지 등 데이터는 Javascript, jQuery를 사용하여 API 호출 작업 수행
  • Excel 파일 다운로드 링크는 Java의 File, Workbook Library로 데이터 추출하여 JSON 파싱 작업

개선작업 및 성과

  • 추후에 특집 페이지 개발 프로젝트를 대비하여 리팩터링 작업 병행
  • application.properties에 개발/상용 구분을 위한 공통 플래그 추가
  • 기존 8개의 파일에 11줄 개별 수정했었으나 properties 내의 플래그 변수 1개로 변환 가능
  • API URL 관리를 위한 통합 작업
  • 기존 17개의 프로그램 개별 URL 사용했으나 API 공통 Script에 모든 API URL 통합 사용
  • 영상의 유형을 구분하는 키와 값의 의미가 고객사에서 사용하는 의미와 통일되도록 전체 개선
  • 기존 소스에 사용하던 용어들의 의미가 혼용된 부분들을 고객사와 동일하게 수정 개선
  • today() 및 각종 Formatting 처리 함수 등 개선 작업

결과

  • 월드컵 개막일까지 설계한 모든 서비스 기능 100% 개발 완료
  • 개발 기간과 서비스 진행 중에 페이지 추가 포함한 별도 요청사항 100% 처리 완료
